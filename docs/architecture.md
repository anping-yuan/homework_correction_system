# 架构设计文档

## 系统概述

作业批改系统采用模块化设计，将采集、预处理、OCR识别、LLM批改和批注渲染分为独立模块，通过主程序协调各模块完成完整流程。

## 架构图

```
┌─────────────┐    ┌────────────────┐    ┌─────────────┐
│  CameraCapture │───▶│ ImageProcessor │───▶│  AliyunOCR    │
│  (图像采集)   │    │  (图像预处理)   │    │  (文字识别)   │
└─────────────┘    └────────────────┘    └──────┬──────┘
                                                │
                                                ▼
┌─────────────────┐                  ┌─────────────────┐
│ AnnotationRenderer │◀──────────────│   LLMGrader       │
│  (批注渲染)        │               │   (智能批改)       │
└─────────────────┘                  └─────────────────┘
```

## 模块说明

### CameraCapture（摄像头采集）
- 基于 OpenCV 实现
- 支持单帧抓拍和连续采集
- 上下文管理器支持（with语句）

### ImageProcessor（图像预处理）
- 灰度化、二值化（OTSU/自适应）
- 去噪处理
- 自动纠偏
- 尺寸调整

### AliyunOCR（阿里云OCR）
- 印刷体文字识别
- 手写文字识别
- 公式识别
- 文字区域定位

### LLMGrader（大模型批改）
- 自定义提示词模板
- 单题/批量批改
- 自动评分
- 评语生成

### AnnotationRenderer（批注渲染）
- 正确/错误标记绘制
- 文字批注叠加
- 分数展示
- 批量批注渲染
